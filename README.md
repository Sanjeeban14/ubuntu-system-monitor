# Ubuntu System Monitor

A terminal-based system monitor written in Bash for Ubuntu Linux.

## Features
 - CPU Usage
 - Memory usage
 - Disk usage
 - System uptime
 - Top 5 CPU-consuming processes
 - Auto-refresh every 2 seconds
 - Colorized outut
